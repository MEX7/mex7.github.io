把状态栏修改成黑色背景

需要在Info.plist配置文件中，增加键：UIViewControllerBasedStatusBarAppearance，并设置为YES；

- (UIStatusBarStyle)preferredStatusBarStyle
{
    return UIStatusBarStyleLightContent;
}
- (BOOL)prefersStatusBarHidden
{
    return NO;
}

Storyboard

### 这是从一个xib跳往名为Main.storyboard的Storyboard，而且我发现跳转storyboard不需要import

<pre>`UIStoryboard *storyboard =[UIStoryboard storyboardWithName:@"Main" bundle:nil];
[self presentViewController:[storyboard instantiateInitialViewController] animated:YES completion:nil];
`</pre>
