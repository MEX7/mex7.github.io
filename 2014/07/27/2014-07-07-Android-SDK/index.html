Android SDK更新过程中的异常

Jul 27, 2014

最近有个安卓程序需要调试。

多次安装都以失败告终

解决方案

如果还有错，可以在文件（C:\Windows\System32\drivers\etc\hosts）中，添加如下几行：

203.208.46.146 www.google.com
203.208.46.146 dl.google.com 
203.208.46.146 dl-ssl.google.com

效果显著。
