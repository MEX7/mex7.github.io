日常打卡 CentOS install Tomcat

Nov 3, 2014

搞定了centos上面的tomcat，在网上看博客各种说下载源码cp后怎样怎样的，当时网速太卡，我就试了下

yum install tomcat

呵呵…居然就这么安装好了，只能说明那些博客太古老了，现在呃yum库都自带了，还是tomcat7

试了下我的想法，把java二进制文件传到bitbucked上，然后再pull到centos上，用着这半手动的方式管理发布。是不是应该再买一个阿里云大家git服务器自己部署hook，这样省事多了，确实是这样的，突然想到bitbucked会不会被墙了，被墙的话，公司财产就都GG了，看来等到2.0版发布之后就应该开始准备这个了，好像是之前确实有点太迷信bitbucked。

安装好tomcat之后就配置路径，再/etc/tomcat/server.xml里面，就不多说了，看例子好像可以设置多域名访问。

简单的在bash里面写了一个alias，用起来肯定比以前用windwos服务器要舒服的多。

vim ~/.bash_profile

最后，还遇到了一个问题，在git push的时候提示

POST git-receive-pack (chunked)

看字面意思就是网络阻塞
解决方法就是在.git/config文件中加入

[http] 
    postBuffer = 524288000

就可以搞定了

最后就是这样，目前的打算就是放弃FTP服务器，使用又拍云进行图像存储。
