Redmine

Aug 7, 2014

ruby 的初学者，对linux系统构建稍微熟悉。

最近在尝试在vps上搭建redmine，有点障碍，再次简略叙述，凭借记忆写点。

  1. 安装mysql，使用RVM安装ruby和rails，之前只安装过一次没有积累足够的经验，磨磨蹭蹭用了半天装完，但不确定是否与最新的redmine匹配
  2. 在github上下载redmine
  3. bundle项目，这里出现的问题比较多，感觉像是现有文件很多gem缺失，后期的感觉就是再给bundle库打补丁
  4. 一切完成发布遇到redmine500错误

至此，我已经用了3天的时间，中间零零碎碎。

开始对rails的理解是集成好了一切的东西，包括数据库，感觉很强大，很强大。在实际使用中由于我个人熟悉程度有限，使用中需要查很多资料东拼西凑完成整个bundle库，感觉没有尽头，所以到发文时刻，我暂时停止，开始写我的党校报告。可能是版本差异导致更新问题，项目的部署需要bundle，但是开发中使用到的一些在我本地库中缺失，基于种种原因，感觉部署的过程是一个杂合再杂合的过程。
