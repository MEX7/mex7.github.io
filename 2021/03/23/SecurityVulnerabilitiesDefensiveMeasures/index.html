SecurityVulnerabilitiesDefensiveMeasures

Mar 23, 2021

SQL注入攻击

  • sql 转换，特殊字符转义，危险字符、关键词过滤
  • 输入字段类型限制、长度限制
  • 统一采用 utf8mb4 编码
  • 限制客户端对数据库的操作权限

XSS（Cross Site Scripting）跨站脚本攻击

本质是 html 注入

  • 字符过滤
  • 服务端进行数据内容校验
  • 白名单屏蔽恶意链接
  • 后端对页面输出进行编码转换

CSRF（Server-Side Request Forgery）跨站请求伪造攻击

请求伪造

  • 服务端进行 Referer 验证
  • 具备过期时间的 token
  • header 加入自定义属性校验，短时间或者单词请求有效

任意文件上传、下载

  • 上传下载身份验证，用户只操作自有权限文件或文件类型
  • 验证文件类型、大小，后端进行文件类型校验、目录执行权限限制
  • 禁止回传绝对路径，禁止用户上传数据进行目录指定和文件名称定义
  • 限制目录权限，提供独立的读取接口

越权访问

  • 数据从当前登录态获取

未授权访问

  • api 接口签名

明文传输

  • 公网走 https
  • 敏感数据单独加密
  • 核心数据存储加密

敏感信息泄露

  • 接口返回数据
  • 注释
  • 程序后门
  • 报错提示

暴力破解

  • 用户登录次数限制
  • IP访问限制
  • 服务端校验
  • API鉴权

XXE（XML External Entity Injection）XML 外部实体注入攻击

  • 对用户提交的 XML 数据进行关键词过滤

短信、邮件轰炸

  • 限制使用频率
  • 异步发送，单次发布记录持久化后进行重复校验
