Apr 24, 2015

前言

最近除了去百度实习之外，我想做的就是这个：

基于python django框架只支持markdwon格式的团队文献管理工具

项目地址

项目在coding上创建好了，ok第一阶段准备工作完成。 我现在也不想想什么股份、合同什么的事情，就想做自己开心的事情，然后明年去实习。

基础开发环境安装

#pip 
sudo easy_install pip

#Django
sudo pip install Django==1.7.7

# 这一步一定要记得sudo，不然就失败了... 这里写了几百字...没保存...掉了...(=ﾟωﾟ)ﾉ（大概讲的就是还有另外一种安装Django的方式...不想写了...）

#app是自己定义的项目名   
django-admin.py startproject app

#启动django，端口可以自己制定 
./manage.py runserver 8087

#建立模块
./manage.py startapp app_editor

完成Hello World，可以睡觉了
